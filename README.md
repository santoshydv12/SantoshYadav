# Git Class Project

This is a simple project created to learn Git and GitHub.

## Files

- `index.html` - Main web page of the project.
- `README.md` - Project documentation.

## Git Commands Used

```bash
git init
git add .
git commit -m "Initial Commit"
git branch -M main
git remote add origin <repository-url>
git push -u origin main
```

## Project Objective

- Learn Git basics
- Learn GitHub repository management
- Practice commits and pushes
- Understand version control
